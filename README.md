This project is a **text-based escape room game in Java**. The player and their friends enter a haunted house escape room on Halloween night with only 10 minutes to escape. By exploring different locations, collecting items, and using commands, the player must solve puzzles that lead to the final code to unlock the door.  

### Features
- Object-oriented design with `Player`, `Friends`, `Locations`, and `Items` classes  
- Interactive commands (e.g., `use remote TV`, `use key safe`, `use code lock`)  
- Branching storylines and multiple endings (escape successfully or run out of time)  
- Emphasis on problem-solving and user interaction
